# 02 - 方法（函数）

## 学习目标

- 理解什么是方法及其作用。
- 掌握如何定义和调用方法。
- 学习方法的参数和返回值。
- 了解方法重载的概念。

## 核心知识点

1.  **方法的定义**:
    -   方法是组织在类中的一段代码，用于执行特定任务。
    -   基本语法: `[修饰符] 返回值类型 方法名([参数列表]) { // 方法体 }`
    -   示例: `public static void myMethod(String name) { ... }`

2.  **参数 (Parameters)**:
    -   方法可以接受零个或多个输入，称为参数。
    -   参数在方法签名中定义，如 `(int a, int b)`。

3.  **返回值 (Return Value)**:
    -   方法可以返回一个值。返回值类型在方法签名中声明。
    -   使用 `return` 关键字返回值。
    -   如果方法不返回任何值，则返回值类型为 `void`。

4.  **方法调用**:
    -   使用方法名和所需的参数来执行方法。
    -   示例: `int sum = add(10, 20);`

5.  **方法重载 (Overloading)**:
    -   在同一个类中，可以有多个同名的方法，但它们的参数列表必须不同（参数的数量、类型或顺序不同）。
    -   编译器会根据你传递的参数来决定调用哪个版本的方法。

## 代码说明

-   `Main.java`: 调用 `MethodExample` 来运行示例。
-   `MethodExample.java`:
    -   `printGreeting()`: 一个简单的方法，没有参数和返回值。
    -   `printPersonalizedGreeting(String name)`: 接受一个字符串参数，但没有返回值。
    -   `add(int a, int b)`: 接受两个整数参数，并返回它们的和。
    -   `add(double a, double b)`: `add` 方法的重载版本，用于处理 `double` 类型的数字。
    -   `main` 方法中包含了对以上所有方法的调用示例。

## 如何运行

1.  打开终端并进入 `02_methods` 目录。
2.  编译所有 Java 文件:
    ```shell
    javac *.java
    ```
3.  运行 `Main` 类:
    ```shell
    java Main
    ```
4.  你将在终端看到方法的调用过程和结果。
