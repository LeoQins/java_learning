# 03 - 面向对象编程 (OOP)

## 学习目标

- 理解类（Class）和对象（Object）的概念。
- 学习如何定义一个类，包括属性（字段）和方法。
- 掌握如何创建和使用对象。
- 理解面向对象的三大特性：封装、继承和多态（本例主要展示前两者）。
- 学习构造方法和方法重写（Override）。

## 核心知识点

1.  **类 (Class)**:
    -   类是创建对象的蓝图或模板。它定义了一类事物的共同属性和行为。
    -   `class Person { ... }`

2.  **对象 (Object)**:
    -   对象是类的实例。
    -   使用 `new` 关键字创建对象: `Person p = new Person("张三", 30);`

3.  **封装 (Encapsulation)**:
    -   将数据（属性）和操作数据的代码（方法）捆绑在一起。
    -   使用 `private` 访问修饰符隐藏内部数据，通过 `public` 的 `getter` 和 `setter` 方法来控制访问，增加了安全性和灵活性。

4.  **继承 (Inheritance)**:
    -   一个类（子类）可以继承另一个类（父类）的属性和方法。
    -   使用 `extends` 关键字实现继承: `class Student extends Person { ... }`
    -   子类可以重用父类的代码，并可以添加自己特有的功能。

5.  **构造方法 (Constructor)**:
    -   一个特殊的方法，用于在创建对象时进行初始化。
    -   方法名与类名相同，没有返回值。
    -   子类的构造方法必须通过 `super()` 调用父类的构造方法。

6.  **方法重写 (Overriding)**:
    -   子类可以提供一个与父类方法签名完全相同的方法，从而覆盖父类的实现。
    -   使用 `@Override` 注解是一个好习惯，可以帮助编译器检查错误。

## 代码说明

-   `Main.java`: 创建 `Person` 和 `Student` 类的对象，并调用它们的方法来演示 OOP 的基本概念。
-   `Person.java`: 一个基础的父类。
    -   它封装了 `name` 和 `age` 属性（私有）。
    -   提供了公共的 `getter/setter` 方法来访问这些属性。
    -   包含一个 `introduce` 方法。
-   `Student.java`: 继承自 `Person` 类。
    -   它继承了 `name` 和 `age`，并添加了自己独有的 `studentId` 属性。
    -   它的构造方法通过 `super()` 调用了父类的构造方法。
    -   它重写了 `introduce` 方法以提供更具体的实现。
    -   它有一个自己的方法 `study()`。

## 如何运行

1.  打开终端并进入 `03_oop` 目录。
2.  编译所有 Java 文件:
    ```shell
    javac *.java
    ```
3.  运行 `Main` 类:
    ```shell
    java Main
    ```
4.  你将看到 `Person` 和 `Student` 对象被创建，以及它们各自方法的输出。
